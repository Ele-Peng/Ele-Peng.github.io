## 双指针

```javascript
let max = 0;
for (let i = 0, j = nums.length - 1; i !== j; ) {
    max = Math.max(max, Math.min(nums[i], nums[j]) * (j - i)) ;
    if (nums[i] < nums[j]) {
        i ++;
    } else {
        j --;
    }
}
return max;
```
